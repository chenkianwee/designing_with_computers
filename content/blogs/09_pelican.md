Title: How I Developed My Website Using Python Pelican

Status: draft
Duration: 5 mins
Category: Tutorial

I have been using Jekyll with Github pages for my website. Jekyll is a great static site generator with many great themes available. I was quite happy with my Jekyll site. However, Jekyll is written in Ruby, I am much more comfortable with Python. I get really nervous about breaking stuff everytime I want to change the underlying theme. I chose Jekyll previously cause of the availability of great looking themes, which is not exactly the case with Pelican. I finally found the time and the determination to make the transition to Pelican recently.
